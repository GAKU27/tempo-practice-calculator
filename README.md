# Tempo Practice Calculator | テンポ練習時間計算機

段階的テンポ上昇トレーニング（ピラミッド練習/ラダー練習）の総所要時間を高精度に算出するWebアプリケーションです。

## 🎵 概要

メトロノームを用いた段階的なテンポ上昇練習において、総所要時間を正確に見積もることは直感的に困難です。テンポと時間は反比例（t ∝ 1/v）の関係にあり、低速域での滞留時間が予想以上に長くなります。

このツールは、数学的に厳密な計算により、練習時間を正確に算出します。

## 🚀 デモ

[GitHub Pagesでデモを見る](https://YOUR_USERNAME.github.io/tempo-practice-calculator/)

## ✨ 機能

- **高精度計算**: Kahan加算アルゴリズムによる浮動小数点誤差の最小化
- **2つの計算手法**: 厳密解と近似解（拡張有理S-P公式）の両方を表示
- **誤差率表示**: 近似解の精度を確認可能（通常+1〜2%以内）
- **レスポンシブデザイン**: モバイル対応のモダンUI

## 📊 入力パラメータ

| パラメータ | 説明 | 例 |
|-----------|------|-----|
| 開始テンポ (a) | 練習開始時のBPM | 60 |
| 目標テンポ (b) | 最終到達BPM | 120 |
| ステップ幅 (s) | 1回の加速量 | 4 |
| フレーズ拍数 (B) | 1フレーズの長さ | 4 |
| 反復回数 (R) | 各テンポでの繰り返し | 4 |
| セット数 (N) | 全体の反復回数 | 1 |

## 🔬 数理モデル

総所要時間は以下の公式で計算されます：

```
T_total = N × Σ(k=0 to n) [60 × B × R / (a + s × k)]
```

ここで：
- n = ⌊(b - a) / s⌋ （ステップ数）
- 厳密解は調和数列の有限部分和
- 近似解は拡張有理S-P公式による
